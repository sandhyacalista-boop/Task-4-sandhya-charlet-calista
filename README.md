# Task-4-sandhya-charlet-calista
Frontend and Backend API Integration Project

Overview

This project demonstrates how to integrate a frontend application with a backend API using HTML, JavaScript, Node.js, and Express.js. The frontend sends requests to the backend, receives JSON data, and dynamically displays it on the user interface.

Features

- Frontend to Backend API communication
- Dynamic data rendering on the webpage
- Asynchronous API requests using Fetch API
- Error handling with Try-Catch
- Cross-Origin Resource Sharing (CORS) support
- Complete Full Stack Data Flow

Technologies Used

Frontend

- HTML5
- JavaScript (ES6)

Backend

- Node.js
- Express.js
- CORS

How It Works

1. User opens the frontend webpage.
2. JavaScript sends a request to the backend API.
3. Backend processes the request and returns JSON data.
4. Frontend receives the response.
5. Dynamic content is displayed on the webpage.
6. Errors are handled and displayed if the request fails.
